クラフトビール記事に旅行アフィリエイトリンクを自動生成するWordPressプラグイン

== Description ==

Beer Affiliate Engineは、クラフトビールに関する記事内の地域名を検出し、その地域への旅行やビール体験に関するアフィリエイトリンクを自動的に生成するプラグインです。

= 主な機能 =

* 楽天トラベル、JTB、読売旅行などの主要旅行サービスのリンク生成
* 一休.comレストラン、Otomoniなどのビール体験系サービス対応
* 管理画面からの簡単設定
* ユーザーフレンドリーなリンク表示

== Installation ==

1. プラグインファイルを `/wp-content/plugins/` ディレクトリにアップロード
2. WordPressの'プラグイン'メニューからプラグインを有効化
3. 管理メニューの「Beer Affiliate」から設定

== Frequently Asked Questions ==

= 対応しているアフィリエイトサービスは？ =

楽天アフィリエイト、A8.netに対応しています。

= ショートコードの使い方は？ =

記事内に [beer_affiliate city="東京"] のように記述してください。
